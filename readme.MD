# Airplane Seating Algorithm

## Tech Spec
* NodeJS
* Mocha JS
* Chai
* Express
* Expresjs

## Build Setup

``` bash
# install dependencies
npm install
npm install mocha -g

# run script
node run.js

# run test script
mocha
```

## Input

Algorithm can be test at (https://armillary-airplane.netlify.com/)[https://armillary-airplane.netlify.com/]

It accepts passengers **(Integer)** and Seats **(2D Array)**
